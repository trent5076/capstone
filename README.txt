To use this data product, follow these instructions:

1. Go to colab.google.reasearch.com
 
2. Sign in or create an account with Google

3. Click on “file” 

4. Select the “upload notebook” option
 
5.Select the “github” option on the popup screen
 
6.Enter the following URL: https://github.com/trent5076/capstone

7.Click the magnifying glass icon
 
8.Click on “Runtime” from the menu at the top

9. Select “Run all” (Or just use keyboard shortcut ctrl + F9)

10. View the code and its outputs (NOTE: Sections of the code are collapsed by default. To expand a section of code, click on the triangle/ arrow icon next to the name of the section you wish to view)
